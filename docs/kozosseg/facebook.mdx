# <FontAwesomeIcon icon="fa-brands fa-square-facebook" /> Facebook

Igyekszünk több Facebook csatornán is jelen lenni, hogy tovább tudjuk adni a WordPress-ben szerzett tudásunkat és tapasztalatainkat, hiszen a WordPress lényege (ahogyan a legtöbb nyílt forráskódú szoftver esetében) a közösség.

### HelloWP! Klub

A Klub oldal nagyszerű példája annak, hogy milyen remek közösség a magyar WordPress-es világ.

https://www.facebook.com/groups/hellowp.klub

### HelloWP! oldal

Ritkán használjuk. Arra való, hogy a fontosabb dolgokról, eseményekről, hírekről tudjunk itt írunk. Mi jobban preferáljuk a Klub oldalt, mert a kétoldalú kommunikáció sokkal hatékonyabb.

https://www.facebook.com/hellowp.io

## További WordPress közösségek a Facebook-on

### WordPress Magyarország

A WordPress Magyarország oldalát a Nyílt Web Alapítvány kezeli. Itt találhatóak a fontosabb WordPress-es történések, események és hírek magyarul.

https://www.facebook.com/wphungary

### WordPress fejlesztők és felhasználók csoportja

A WordPress magyar közösségének a szíve és a lelke ez a csoport. Jelenleg közel 13.000 tagja van.

https://www.facebook.com/groups/wpcsoport/

### wpBarkács

Buxbaum Barna és a WPSuli csapata a WordPress tanulásához és a WordPress fejlesztőképzéshez készített oldal.

https://www.facebook.com/groups/1274367906000411
