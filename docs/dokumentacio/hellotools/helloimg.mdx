**Weboldal**: [https://helloimg.io](https://helloimg.io)

## Használat WordPress-ben

A HelloIMG szolgáltatás a népszerű képoptimalizáló bővítményeken keresztül használható:

1. Telepítsd az **Imagify** vagy **ShortPixel** bővítményt
2. A HelloPack kliensben kapcsold be a megfelelő modult:
   - **WordPress admin** → **HelloPack** → **Modulok**
   - Kapcsold be az **Imagify** vagy **ShortPixel** modult
3. A bővítmény ezután a HelloIMG szolgáltatást fogja használni a képoptimalizáláshoz

:::tip Előny
Nincs szükség külön Imagify vagy ShortPixel előfizetésre – a HelloIMG API-t használod a HelloPack előfizetéseddel, korlátlan oldalon és korlátlan képpel.
:::

:::note Próbáld ki ingyen!
A [helloimg.io](https://helloimg.io) oldalon regisztráció nélkül tesztelheted a szolgáltatást. Töltsd fel a képeidet és győződj meg a minőségről!
:::

## Főbb funkciók

### <FontAwesomeIcon icon="fa-duotone fa-images" /> Univerzális formátum támogatás

A HelloIMG minden képformátumot kezel: JPEG, PNG, GIF és WebP. Legyen szó termékfotókról, blog képekről vagy marketing anyagokról.

### <FontAwesomeIcon icon="fa-duotone fa-rocket" /> Újgenerációs formátum konverzió

Automatikus konverzió modern formátumokba (WebP, AVIF) a maximális tömörítésért és villámgyors betöltésért. Javítsd a Core Web Vitals mutatóidat és a Google PageSpeed pontszámod.

### <FontAwesomeIcon icon="fa-duotone fa-wand-magic-sparkles" /> Intelligens tömörítés

Nincs szükség beállításokra. Az intelligens tömörítő motor elemzi minden képet és automatikusan alkalmazza az optimális beállításokat. Tökéletes eredmény minden alkalommal.

### <FontAwesomeIcon icon="fa-duotone fa-arrows-maximize" /> Rugalmas átméretezés

Képek dinamikus átméretezése az API-n keresztül. Egyedi méretek, képarány megtartás vagy százalékos skálázás. Tökéletes reszponzív designhoz.

### <FontAwesomeIcon icon="fa-duotone fa-code" /> API-First Design

Egyszerű REST API átfogó dokumentációval. Adj hozzá képoptimalizálást a munkafolyamatodhoz néhány sor kóddal. Node.js, Python, PHP – mindet támogatjuk.

### <FontAwesomeIcon icon="fa-duotone fa-bolt" /> Valós idejű feldolgozás

Feltöltés és optimalizálás másodpercek alatt. Nincs várakozási sor. Azonnal megkapod az optimalizált képeket.

## API integráció

Kezdj el percek alatt az egyszerű REST API-val:

```bash title="cURL példa"
curl -X POST https://api.helloimg.io/api/upload \
  -H "Authorization: token YOUR_API_KEY" \
  -F "image=@photo.jpg" \
  -F "data[ultra]=true" \
  -F "data[convert]=webp"
```
