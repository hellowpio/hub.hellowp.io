# <FontAwesomeIcon icon="fa-sharp fa-solid fa-envelopes-bulk text-primary" /> Levelezés beállítása Microsoft Outlook-ban

Az Outlook automatikusan felismeri a MailHub szerver beállításait, így a legtöbb esetben csak az email címed és jelszavad megadása szükséges.

## Outlook (Microsoft 365 / 2019 / 2021) - Windows

1. Indítsd el az **Outlook**-ot
2. Ha először indítod, megjelenik a fiók beállító varázsló. Ha már van fiókod, menj a **Fájl → Fiók hozzáadása** menüpontra
3. Add meg az **email címedet** és kattints a **Csatlakozás** gombra
4. Add meg a **jelszavadat**, majd kattints a **Bejelentkezés** gombra
5. Ha a rendszer kéri, engedélyezd a hozzáférést az **Engedélyezés** gombra kattintva
6. Kattints a **Kész** gombra

:::tip Automatikus konfiguráció
Az Outlook az autodiscover funkció segítségével automatikusan megtalálja a megfelelő szerver beállításokat. Ha ez nem működik, használd a [manuális beállítás](./levelezes-beallitasa) útmutatóját.
:::

## Outlook - macOS

1. Indítsd el az **Outlook**-ot
2. Menj az **Outlook → Beállítások → Fiókok** menüpontra
3. Kattints a **+** gombra és válaszd az **Új fiók** opciót
4. Add meg az **email címedet** és kattints a **Folytatás** gombra
5. Add meg a **jelszavadat** és kattints a **Fiók hozzáadása** gombra

:::info Naptár és névjegyek szinkronizálása
A macOS Outlook verzió alapértelmezetten nem szinkronizálja a naptárakat és névjegyeket. Ehhez használd a rendszer beépített **Mail**, **Naptár** és **Névjegyek** alkalmazásait az [Apple beállítás útmutató](./apple-macos-ios) szerint.
:::

## Outlook Mobile (iOS / Android)

1. Töltsd le az **Outlook** alkalmazást az App Store-ból vagy Google Play-ből
2. Nyisd meg az alkalmazást és koppints a **Fiók hozzáadása** gombra
3. Add meg az **email címedet** és koppints a **Folytatás** gombra
4. Add meg a **jelszavadat** és koppints a **Bejelentkezés** gombra
5. Engedélyezd a szükséges jogosultságokat

:::tip Push értesítések
Az Outlook mobilalkalmazás támogatja a push értesítéseket, így azonnal értesülsz az új levelekről.
:::
