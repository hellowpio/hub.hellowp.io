# <FontAwesomeIcon icon="fa-sharp fa-solid fa-envelopes-bulk text-primary" /> Levelezés beállítása Windows Mail-ben

A Windows 10 és 11 beépített Mail alkalmazása támogatja az Exchange ActiveSync protokollt, így egyszerűen beállíthatod a MailHub fiókodat.

## Fiók hozzáadása

1. Nyisd meg a **Mail** alkalmazást a Start menüből
2. Ha először indítod, kattints a **Fiók hozzáadása** gombra. Ha már van fiókod, kattints a bal oldalsáv alján található **Beállítások** (fogaskerék) ikonra, majd válaszd a **Fiókok kezelése → Fiók hozzáadása** opciót
3. Válaszd az **Exchange** lehetőséget
4. Add meg az **email címedet** és kattints a **Tovább** gombra
5. Add meg a **jelszavadat** és kattints a **Bejelentkezés** gombra
6. Kattints a **Kész** gombra

:::tip Naptár és Névjegyek
A Mail alkalmazással együtt automatikusan beállításra kerül a **Naptár** és a **Személyek** (névjegyek) alkalmazás is, így ezeket azonnal használhatod.
:::

## Szinkronizálási beállítások

A szinkronizálás gyakoriságát módosíthatod:

1. Nyisd meg a **Beállítások** → **Fiókok kezelése** menüt
2. Kattints a MailHub fiókodra
3. Válaszd a **Postafiók szinkronizálási beállításainak módosítása** opciót
4. Állítsd be a kívánt szinkronizálási gyakoriságot

:::info Windows 11
Windows 11-en az új Outlook alkalmazás érhető el, amely fokozatosan felváltja a hagyományos Mail alkalmazást. Az új Outlook esetében kövesd a [Microsoft Outlook](./microsoft-outlook) útmutatót.
:::
