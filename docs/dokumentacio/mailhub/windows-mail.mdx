# <FontAwesomeIcon icon="fa-sharp fa-solid fa-envelopes-bulk text-primary" /> Levelezés beállítása Windows Mail-en
A Windows 8 és magasabb verziók támogatják az emailt, névjegyeket és a naptárat az Exchange ActiveSync-en keresztül.


1. Nyisd meg a Mail alkalmazást.
2. Ha még nem használtad korábban a Mailt, kattinthatsz az Add Account gombra a fő ablakban. Lépj tovább a 4. lépésre.
3. Kattints a Bal oldali oldalsávon található Accounts-ra, majd kattints a jobb oldalon található Add Account-ra.
4. Válaszd az Exchange opciót.
5. Add meg az email címed, majd kattints a Next-re.
6. Add meg a jelszavad, majd kattints a Log in-re.

Miután beállítottad a Mail alkalmazást, használhatod a People és a Calendar alkalmazásokat is.
