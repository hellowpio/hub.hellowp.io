## Fiók adatok

| Beállítás | Érték |
|-----------|-------|
| **Felhasználónév** | Az email címed |
| **Jelszó** | Email-ben küldtük, vagy amit megadtál |

## Szerver beállítások

### POP3 (bejövő levelek)

| Beállítás | Érték |
|-----------|-------|
| **Szerver** | mail.webmi.io |
| **Port** | 995 |
| **Titkosítás** | SSL/TLS |
| **Hitelesítés** | Normál jelszó |

### IMAP (bejövő levelek - ajánlott)

| Beállítás | Érték |
|-----------|-------|
| **Szerver** | mail.webmi.io |
| **Port** | 993 |
| **Titkosítás** | SSL/TLS |
| **Hitelesítés** | Normál jelszó |

### SMTP (kimenő levelek)

| Beállítás | Érték |
|-----------|-------|
| **Szerver** | mail.webmi.io |
| **Port** | 465 |
| **Titkosítás** | SSL/TLS |
| **Hitelesítés** | Szükséges |

:::tip IMAP vs POP3
Az **IMAP** használatát javasoljuk, mert a levelek a szerveren maradnak és több eszközről is elérheted őket. A POP3 letölti a leveleket az eszközre és (alapértelmezetten) törli a szerverről.
:::

## Ajánlott levelezőprogramok

| Platform | Ajánlott kliens |
|----------|-----------------|
| Windows | Outlook, Thunderbird |
| macOS | Apple Mail, Thunderbird |
| iOS | Apple Mail |

:::warning Figyelem!
A HelloWP levelezőszervere **nem használható hírlevélküldésre** vagy más tömeges kiküldésre. Erre is van kész megoldásunk - vedd fel velünk a kapcsolatot.
:::
