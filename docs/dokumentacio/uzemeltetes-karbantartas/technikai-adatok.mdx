## Szerver infrastruktúra

| Jellemző | Részletek |
|----------|-----------|
| **Szerverek helye** | Európa, Amerika, Ázsia, Ausztrália |
| **Háttértároló** | NVMe SSD |
| **Webszerver** | NGINX (alapértelmezett) |
| **Alternatív webszerver** | NGINX + Apache hibrid (.htaccess támogatás) |

:::info Szerver kiválasztás
Mindig abba a szerverközpontba helyezzük a weboldalt, amelyhez közelebb vannak a látogatóid. Ez biztosítja a lehető leggyorsabb betöltési időt.
:::

## PHP verziók

| Verzió | Állapot |
|--------|---------|
| PHP 8.3 | Elérhető |
| PHP 8.2 | Elérhető (ajánlott) |
| PHP 8.1 | Elérhető |
| PHP 8.0 | Elérhető |
| PHP 7.4 | Elérhető (elavult) |

:::warning PHP 7.4
A PHP 7.4 biztonsági támogatása megszűnt. Javasoljuk a PHP 8.1+ verzióra való frissítést.
:::
