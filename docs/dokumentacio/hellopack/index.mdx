# <FontAwesomeIcon icon="fa-duotone fa-parachute-box text-success" /> HelloPack

A HelloPack csomagkezelő bővítmény segítségével egyszerűen kezelheted a prémium WordPress bővítményeket és témákat. A csomagkezelő biztosítja a frissítéseket, aktiválásokat és deaktiválásokat.

## Főbb funkciók

- **Központosított bővítménykezelés** – Kezeld az összes WordPress oldalad bővítményeit egy helyről
- **Verziózás** – Kövesd nyomon a bővítmények verzióit és frissítési előzményeit
- **Gyors telepítés** – Telepíts prémium bővítményeket és témákat egyetlen kattintással
- **Frissítések** – Automatikus értesítések és egyszerű frissítési folyamat
- **Tanácsadás és támogatás** – Szakértői segítség a bővítményekkel kapcsolatos kérdésekben

## Hogyan működik?

### 1. HelloPack kliens telepítése

Telepítsd a HelloPack kliens bővítményt a WordPress oldaladra:

1. Töltsd le a klienst a [HelloConsole](https://hellowp.io/hu/helloconsole/hellopack-kozpont/)-ból
2. Telepítsd a WordPress admin → Bővítmények → Új hozzáadása → Bővítmény feltöltése menüpontból

### 2. API kulcs aktiválása

Aktiváld a klienst az API kulcsoddal a WordPress adminban vagy parancssorból:

```bash
wp hellopack license activate <api-kulcs>
```

### 3. Bővítmények kezelése

A [HelloConsole](https://hellowp.io/hu/helloconsole/) felületén vagy közvetlenül a WordPress adminban kezelheted a bővítményeket.
