# <FontAwesomeIcon icon="fa-duotone fa-server text-primary" /> Rendszerkövetelmények

A HelloPack követelményei megegyeznek a [WordPress követelményeivel](https://wordpress.org/about/requirements/).

## Szoftverkövetelmények

| Komponens | Minimum | Ajánlott |
|-----------|---------|----------|
| **PHP** | 7.4 | 8.1+ |
| **MySQL** | 8.0 | 8.0+ |
| **MariaDB** | 10.4 | 10.6+ |
| **WordPress** | 5.8 | 6.4+ |

## Webszerver

A HelloPack az alábbi webszerverekkel működik:

- **Apache** (ajánlott)
- **Nginx**
- **LiteSpeed**

:::note LiteSpeed beállítások
LiteSpeed-et csak akkor ajánljuk, ha jól be van állítva. Gyakori hiba, hogy a rosszul beállított LiteSpeed cache alatt tartja az admin felületet, így pl. az Elementor nem fut megfelelően. Győződj meg róla, hogy az admin felület ki van zárva a cache-ből.
:::

## PHP kiterjesztések

A következő PHP kiterjesztések szükségesek:

- `curl`
- `zip` (ZipArchive)
- `json`
- `mbstring`

## HTTPS

A HelloPack használatához **HTTPS kapcsolat szükséges**. Győződj meg róla, hogy a weboldalad SSL tanúsítvánnyal rendelkezik.
