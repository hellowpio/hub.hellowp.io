# <FontAwesomeIcon icon="fa-duotone fa-arrows-rotate text-primary" /> Bővítmények frissítése

A HelloPack-os bővítmények frissítése ugyanúgy működik, mint bármely más WordPress bővítmény frissítése.

## Frissítés módjai

### 1. Telepített bővítmények oldalon

1. Menj a **WordPress admin → Bővítmények → Telepített bővítmények** menüpontra
2. A frissíthető bővítményeknél megjelenik az **„Új verzió érhető el"** értesítés
3. Kattints a **Frissítés most** linkre

### 2. Frissítések oldalon

1. Menj a **WordPress admin → Vezérlőpult → Frissítések** menüpontra
2. Itt látod az összes elérhető frissítést (bővítmények, témák, WordPress)
3. Jelöld be a frissíteni kívánt elemeket
4. Kattints a **Bővítmények frissítése** gombra

### 3. Automatikus frissítés

A WordPress lehetővé teszi az automatikus frissítések bekapcsolását:

1. Menj a **Bővítmények → Telepített bővítmények** oldalra
2. A bővítmény mellett kattints az **Automatikus frissítések engedélyezése** linkre

:::tip Frissítés előtt
Készíts biztonsági mentést a weboldaladról, mielőtt frissítenél. Ez különösen fontos nagyobb verziófrissítéseknél.
:::
