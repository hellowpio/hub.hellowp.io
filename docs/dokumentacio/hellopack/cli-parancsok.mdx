## Mi az a WP CLI?

A WP-CLI egy parancssoros interfész (CLI) WordPress-hez. A WP-CLI lehetővé teszi, hogy a WordPress-t parancssorban használd, és könnyen végrehajtsd a feladatokat, amelyeket általában a WordPress adminisztrációs felületén keresztül végeznél. A WP-CLI használatával könnyen telepíthetsz, frissíthetsz, konfigurálhatsz és karbantarthatod a WordPress weboldaladat parancssoros interfész segítségével. A WP-CLI telepítése és használata több platformon is elérhető, beleértve a Windows, a macOS és a Linux rendszereket is.

## HelloPack CLI parancsok

A plugin 3 fő parancskategóriát biztosít:

1. **Licenc kezelés** (`wp hellopack license`)
2. **Plugin kezelés** (`wp hellopack plugin`)
3. **Téma kezelés** (`wp hellopack theme`)

---

## Licenc kezelés (wp hellopack license)

### License aktiválása

Aktiválja a HelloPack licencét a megadott licenckulccsal.

**Paraméterek**

- `<license-key>` - A HelloPack licenckulcsa.

```bash
wp hellopack license activate <license-key>
```

### License deaktiválása

Törli a kulcsot a weboldalról és a hellowp.io szerveréről egyaránt.

```bash
wp hellopack license deactivate
```

### License státusz lekérdezése

Lekéri a HelloPack licenc állapotát.

```bash
wp hellopack license status
```

## Plugin kezelés (wp hellopack plugin)

### Elérhető pluginok listázása

Listázza az összes elérhető plugint, amit a HelloPack-kal telepíthetsz.

```bash
wp hellopack plugin list
```

JSON formátumban:

```bash
wp hellopack plugin list --format=json
```

### Plugin telepítése

Telepíti a megadott plugint slug vagy ID alapján.

**Paraméterek**

- `<slug|id>` - A plugin slug neve vagy ID-ja.
- `--activate` - (opcionális) Telepítés után aktiválja a plugint.

```bash
wp hellopack plugin install elementor-pro
```

Telepítés és aktiválás egyben:

```bash
wp hellopack plugin install 123 --activate
```

### Plugin frissítése

Frissíti a megadott plugint a legújabb verzióra.

**Paraméterek**

- `<slug>` - A plugin slug neve.

```bash
wp hellopack plugin update elementor-pro
```
