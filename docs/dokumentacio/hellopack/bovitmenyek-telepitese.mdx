# <FontAwesomeIcon icon="fa-duotone fa-transporter-3 text-primary" /> Bővítmények telepítése

A HelloPack-ban lévő bővítmények telepítése ugyanolyan egyszerű, mint bármely más WordPress bővítményé.

## Bővítmények telepítése

A bővítményeket egyszerűen és könnyedén tudod telepíteni közvetlenül a Hellopack csoamgkezelővel.

### Lépések

1. Jelentkezz be a WordPress admin felületére
2. A bal oldali menüben keresd meg a **HelloPack** menüpontot
3. Kattints a HelloPack menüpontra, amely megnyitja a HelloPack irányítópultot, itt egyből a **Bővítmények** menüpontra kerülsz (amennyiben már aktiváldat az API kulcsod)

**A HelloPack menüpontban a következő lehetőségeket találod:**

- **Bővítmények **: Itt láthatod az összes elérhető bővítményt
- **Sablonok**: Itt találod a letölthető sablonokat (Bricks)
- **Fordítások**: Itt találod a Hellopack által biztosított magyar fordításokat
- **Beállítások**: Itt tudod ellenőrizni, hogy a kulcsod aktív-e, valamint a kapcsolatot ellenőrizni ha szükséges
- **Modulok**: Itt tudod ki-be kapcsolni a Hellopack belső moduljait, valamint a bővímtény modulokat
- **Tesztelő**: Hiba esetén egy gyors ellenőrző listát kapsz a legalapvetőbb információkkal, amit egyből vágólapra is másolhatsz
