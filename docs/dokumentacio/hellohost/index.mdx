# <FontAwesomeIcon icon="fa-duotone fa-server text-primary" /> HelloHost

A HelloHost a mi menedzselt WordPress tárhely és szerverkezelő platformunk. Ebben a szekcióban a fejlesztői és üzemeltetői használathoz szükséges tudnivalókat gyűjtjük össze: SSH hozzáférés beállítása, parancssori eszközök, automatizáció.

## Gyors áttekintés

- **Hozzáférés**: a HelloHost felülete a [cloud.hellohost.io](https://cloud.hellohost.io) címen érhető el.
- **SSH**: minden kiosztott szerverhez kulcs-alapú SSH hozzáférés jár. Saját SSH kulcsot feltölthetsz a felületen.
- **Alkalmazások**: WordPress, generic PHP és staging környezetek kezelése.
- **File Manager**: beépített web-alapú fájlkezelő SSH nélkül is.
