# <FontAwesomeIcon icon="fa-duotone fa-paintbrush-pencil text-primary" /> Sablon telepítési hiba

WordPress sablon telepítése közben az alábbi hibaüzenetet kapod, ha megpróbálsz egy nem WordPress sablont telepíteni:

```
Csomag kitömörítése…

A sablon telepítése.…

A csomag nem telepíthető. A sablon nem tartalmazza a style.css fájlt.

Sablon telepítési hiba.
```

## Megoldás

Ellenőrizd a ZIP fájlt, hogy az pontosan micsoda. Gyakori hiba, hogy WordPress bővítményt próbálnak telepíteni sablonként. Bővítmény telepítéséhez navigálj a WordPress admin felületen a **Bővítmények** menüpontra, majd kattints a **Új bővítmény hozzáadása** gombra.
