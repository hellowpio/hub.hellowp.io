Ebben az útmutatóban megtudhatod, hogy hogyan költöztethetsz el egy oldalt az **All-in-One WP Migration** bővítménnyel, URL cím alapján (az oldal mentett verziója URL címről letölthető, amit felmásolunk a cél tárhelyre).

## 1. Előkészületek

1. Győződj meg róla, hogy a **cél oldal tárhelyére** (ahová költöztetni szeretnéd az oldalt) feltelepítetted a **WordPress**-t.
2. Telepítsd **mindkét** weboldalra (ahonnan költöztetsz, illetve ahová költöztetnél) az **All-in-One WP Migration** bővítményt.
3. A **cél oldalon** (ahová költöztetni szeretnéd az oldalt) telepítsd fel az **All-in-One WP Migration URL Extension** ([Letöltés](https://cdn.hellowp.cloud/downloads/hellowp-url-extension.zip)) bővítményt.
4. A **cél oldalon** (ahová költöztetni szeretnéd az oldalt) a **Bővítmények** menüponton belül válaszd ki a **Telepített bővítmények** menüpontot.
5. Keresd meg az **All-in-One WP Migration URL Extension** bővítményt, majd kattints a **Frissítések ellenőrzése** szövegre, majd frissítsd a bővítményt.

## 2. Weboldal mentése

A **költöztetendő** oldalon létre kell hoznunk egy mentést az **All-in-One WP Migration** bővítménnyel (ezt a mentést fogjuk URL cím alapján letölteni és felmásolni a tárhelyre), ehhez kövesd a következő lépéseket:

1. A költöztetendő oldal vezérlőpultjának bal oldali sávjában vidd az egeret az **All-in-One WP Migration** menüpontra, majd kattints a **Mentések** menüpontra.
2. Kattints a **Biztonsági mentés készítése** gombra.
3. Amint készen van a biztonsági mentés, kattints a **bezárás** menüpontra.
4. Kattints a biztonsági mentés nevétől jobbra található 3 db egymás alatt lévő **pont ikonra,**
5. Vidd az egeret a **letöltés** gombra (de ne kattints rá a bal egérgombbal)
6. Kattints a **jobb egérgombbal** és válaszd ki a (az általad használt böngészőtől függ, melyik feliratra kell kattintanod) Chrome esetén: **Link címének másolása** / Firefox esetén: **Hivatkozás másolása** / Opera esetén: **Hivatkozás címének másolása** opciót.

Ha ezzel megvagyunk, úgy sikeresen létrehoztunk egy biztonsági mentést, majd kimásoltuk a biztonsági mentés URL címét, amit bárhová beilleszthetünk a **CTRL + V** billentyűkombinációval, vagy a jobb egérgomb lenyomása után a **beillesztés** opcióra kattintva.

7. Nyiss meg egy jegyzettömböt, majd másold be az URL címet a fent említett opciók egyikével (pl.: CTRL + V).

## 3. Weboldal importálása

1. A **cél oldal** vezérlőpultjában vidd az egeret az **All-in-One WP Migration** menüpontra, majd válaszd az **Importálás** opciót.
2. Kattints az **Importálás innen** gombra, majd válaszd ki az **URL** opciót.

5. Tekerj le az oldal aljára és kattints a **Módosítások mentése** gombra.
