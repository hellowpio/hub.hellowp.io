# <FontAwesomeIcon icon="fa-sharp fa-regular fa-code text-primary" /> SMS értesítés Amelia bővítményben

Az SMS értesítés beállításához létre kell hoznod egy ingyenes fiókot az Amelia adminisztrációs felületén. **WordPress admin -> Amelia -> Értesítések -> SMS értesítések -> Feliratkozás** -> Ide írd be az email címedet és a jelszavadat (tetszőleges címet adhatsz meg). Kész is!

Miután létrehoztad az ingyenes fiókodat, fel tudod tölteni a fiókodhoz tartozó egyenleget. Ezt az egyenleget fogja a rendszer használni a küldéshez.
