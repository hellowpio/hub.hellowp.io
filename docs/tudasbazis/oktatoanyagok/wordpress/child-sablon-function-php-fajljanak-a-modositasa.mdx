# <FontAwesomeIcon icon="fa-duotone fa-function text-primary" /> Child sablon function.php fájljának a módosítása

Child sablon használatával a fősablont nyugodtan frissítheted, hiszen a frissítés nem írja felül a child sablonban történt módosításokat. Amikor egy WordPress snippetet szeretnél használni, akkor a child sablon function.php fájljába kell beillesztened.

:::note Mi az a WordPress snippet?

A WordPress snippet egy rövid kódrészlet, amely a WordPress weboldalának különböző funkcióit vagy megjelenését módosítja. A snippeteket általában a functions.php fájlba, vagy egy saját plugin fájljába lehet beilleszteni, és módosíthatják az oldal kinézetét, a használható funkcionalitást, az adatbázis kezelését vagy a felhasználói élményt.
:::

## Hogyan módosítsd a child sablon function.php fájlját?

1. Keresd meg a child sablon function.php fájlját a File Managerben.
2. Kattints a fájlra, majd a fájl jobb felső sarkában lévő **Szerkesztés** gombra.
3. Illeszd be a snippetet a fájl végére.
4. Kattints a **Mentés** gombra.

:::success Tipp
Gyakran módosítod a WordPress fájlokat? Akkor érdemes a [VS Code](https://code.visualstudio.com/) kódszerkesztőt használni.
:::
