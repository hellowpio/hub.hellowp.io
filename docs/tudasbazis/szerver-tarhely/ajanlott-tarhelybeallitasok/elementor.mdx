# <FontAwesomeIcon icon="fa-brands fa-elementor text-primary" /> Elementor ajánlott tárhelybeállítások

Ezen az oldalon összefoglaljuk a rendszerkövetelményeket, amelyek szükségesek az Elementor megfelelő működéséhez. Ha nem vagy biztos benne, hogy a szervered támogatja-e ezeket, kérdezd meg a tárhelyszolgáltatódat.

## PHP beállítások

Az Elementor megfelelő működéséhez az alábbi PHP beállításokat javasoljuk:

```ini title="php.ini"
upload_max_filesize = 128M
post_max_size = 128M
max_execution_time = 300
max_input_time = 300
max_input_vars = 10000
memory_limit = 1024M
```

### Beállítás .htaccess fájlban

Ha nincs hozzáférésed a php.ini fájlhoz, a `.htaccess` fájlban is megadhatod:

```apache title=".htaccess"
php_value upload_max_filesize 128M
php_value post_max_size 128M
php_value max_execution_time 300
php_value max_input_time 300
php_value max_input_vars 10000
php_value memory_limit 1024M
```

:::tip Tipp
Komplex oldalak esetén (sok widget, dinamikus tartalom) növeld a `max_execution_time` értékét 600-ra a timeout hibák elkerülése érdekében.
:::

### Bizonytalan vagy? Nem tudod, hogy ezt hogyan kell beállítani?

Semmi gond, a szolgáltatód tud segíteni. Küldd el nekik az alábbi levelet:

> Tisztelt Szolgáltatóm,
>
> Kérem, állítsák be a domain.hu *(írd át a sajátodra)* tárhelyhez tartozó PHP limiteket az alábbiakra:
>
> - upload_max_filesize: 128M
> - post_max_size: 128M
> - max_execution_time: 300
> - max_input_time: 300
> - max_input_vars: 10000
> - memory_limit: 1024M
>
> Indoklás: A WordPress és Elementor megfelelő működéséhez ezekre a beállításokra van szükségem.
>
> Köszönöm!

## WordPress verzió

**WordPress 6.4 vagy újabb** verzió szükséges az Elementor legújabb verziójához.

:::info Ajánlás
Az Elementor 3.19+ verzióhoz WordPress 6.3 vagy újabb szükséges. Mindig a legújabb WordPress verziót használd.
:::

## PHP verzió

**PHP 8.1 vagy újabb** verzió ajánlott.

| PHP verzió | Támogatás |
|------------|-----------|
| PHP 8.3 | Teljes támogatás |
| PHP 8.2 | Teljes támogatás |
| PHP 8.1 | Teljes támogatás |
| PHP 8.0 | Támogatott, de nem ajánlott |
| PHP 7.4 | Elavult, frissítés szükséges |

:::warning Fontos
A PHP 7.4 biztonsági támogatása 2022 novemberében megszűnt. Az Elementor 3.19+ verzió minimum PHP 7.4-et igényel, de erősen javasolt a PHP 8.1+ használata.
:::

## Adatbázis

- **MySQL 8.0** vagy újabb (ajánlott)
- **MySQL 5.7** minimum
- **MariaDB 10.4** vagy újabb

## WP/PHP memória limit

Az Elementor szerkesztő memóriaigényes alkalmazás. A megfelelő működéshez az alábbi memória beállítások szükségesek:

| Konfiguráció | Minimum memória | Ajánlott memória |
|--------------|-----------------|------------------|
| Csak Elementor | 256 MB | 512 MB |
| Elementor + WooCommerce | 512 MB | 768 MB |
| Elementor + WooCommerce + JetPlugins | 768 MB | 1024 MB |
| Nagy webáruház, komplex oldal | 1024 MB | 1536 MB |

:::note Megjegyzés
Az 1024 MB-os memória limit magas érték, amit nem minden tárhelyszolgáltató támogat. Mielőtt tárhelyet választasz, mindenképpen kérdezd meg, hogy ezt biztosítani tudják-e.
:::

### Memória beállítása wp-config.php-ban

```php title="wp-config.php"
define('WP_MEMORY_LIMIT', '1024M');
```

## Böngésző követelmények

Az Elementor szerkesztő a böngészőben fut, ezért fontos a megfelelő böngésző használata:

| Böngésző | Szerkesztő | Frontend (desktop) | Frontend (mobil) |
|----------|------------|-------------------|------------------|
| Google Chrome | 115+ | 115+ | 115+ |
| Microsoft Edge | 115+ | 115+ | 115+ |
| Firefox | 115+ | 115+ | 115+ |
| Safari | 16+ (macOS) | 16+ (macOS) | 16+ (iOS) |
| Brave | 115+ | 115+ | 115+ |

:::caution Figyelem
Az Elementor szerkesztő **csak asztali számítógépeken** működik. Mobiltelefonon és tableten nem lehet vele szerkeszteni.
:::

## Eszközkövetelmények a szerkesztéshez

A szerkesztő megfelelő működéséhez:

- **Minimum 4 GB RAM** a számítógépben
- **Stabil internetkapcsolat** (legalább 10 Mbps)
- **Hardveres gyorsítás** engedélyezve a böngészőben

:::tip Teljesítmény tipp
Ha lassú a szerkesztő, próbáld meg:
1. Böngésző cache ürítése
2. Böngésző bővítmények kikapcsolása
3. Hardveres gyorsítás engedélyezése
4. Más böngésző használata (Chrome ajánlott)
:::

## Elementor rendszerinformáció

Az Elementor beépített rendszerinformáció oldallal rendelkezik, ahol ellenőrizheted a szerver beállításait:

1. Menj az **Elementor → Rendszerinformáció** menüpontra
2. Ellenőrizd a piros felkiáltójeleket - ezek problémát jeleznek
3. A "Másolás" gombbal kimásolhatod az információkat hibakereséshez

## Gyorsítótár beállítások

Az Elementor jól működik az alábbi cache megoldásokkal:

- WP Rocket
- LiteSpeed Cache
- W3 Total Cache
- Autoptimize

:::caution Fontos cache beállítások
- Kapcsold ki a cache-t bejelentkezett felhasználóknak
- Add hozzá az Elementor szerkesztő URL-jeit a kizárások listájához
- Ne minifikáld az inline CSS/JS-t, amit az Elementor generál
:::
