# <FontAwesomeIcon icon="fa-duotone fa-wind text-primary" /> Kadence ajánlott tárhelybeállítások

Ezen az oldalon összefoglaljuk a rendszerkövetelményeket, amelyek szükségesek a Kadence téma és Kadence Blocks megfelelő működéséhez. Ha nem vagy biztos benne, hogy a szervered támogatja-e ezeket, kérdezd meg a tárhelyszolgáltatódat.

## PHP beállítások

A Kadence megfelelő működéséhez az alábbi PHP beállításokat javasoljuk:

```ini title="php.ini"
upload_max_filesize = 128M
post_max_size = 128M
max_execution_time = 300
max_input_time = 300
max_input_vars = 10000
memory_limit = 256M
```

### Beállítás .htaccess fájlban

Ha nincs hozzáférésed a php.ini fájlhoz, a `.htaccess` fájlban is megadhatod:

```apache title=".htaccess"
php_value upload_max_filesize 128M
php_value post_max_size 128M
php_value max_execution_time 300
php_value max_input_time 300
php_value max_input_vars 10000
php_value memory_limit 256M
```

:::tip Tipp
A Kadence Blocks könnyű és optimalizált, így alacsonyabb erőforrásigényű, mint más page builderek. A fenti beállítások a legtöbb esetben elegendőek.
:::

## WordPress verzió

**WordPress 6.4 vagy újabb** verzió szükséges a Kadence legújabb verziójához.

:::info Blokk szerkesztő
A Kadence Blocks a WordPress natív blokk szerkesztőjét (Gutenberg) bővíti ki. A legjobb élményért mindig a legújabb WordPress verziót használd.
:::

## PHP verzió

**PHP 8.1 vagy újabb** verzió ajánlott.

| PHP verzió | Támogatás |
|------------|-----------|
| PHP 8.3 | Teljes támogatás |
| PHP 8.2 | Teljes támogatás |
| PHP 8.1 | Teljes támogatás |
| PHP 8.0 | Támogatott |
| PHP 7.4 | Elavult, frissítés szükséges |

:::warning Fontos
A PHP 7.4 biztonsági támogatása 2022 novemberében megszűnt. Erősen javasolt a PHP 8.1 vagy újabb verzióra való frissítés.
:::

## Adatbázis

- **MySQL 8.0** vagy újabb (ajánlott)
- **MySQL 5.7** minimum
- **MariaDB 10.4** vagy újabb

## Webszerver

A Kadence az alábbi webszervereken működik:

- **Apache 2.4** (mod_rewrite szükséges)
- **Nginx**
- **LiteSpeed**

## Kadence termékek

A Kadence ökoszisztéma több terméket tartalmaz:

| Termék | Leírás |
|--------|--------|
| **Kadence Theme** | Gyors, könnyű WordPress téma |
| **Kadence Blocks** | Gutenberg blokk bővítmény |
| **Kadence Starter Templates** | Előre elkészített sablonok |
| **Kadence WooCommerce Email Designer** | WooCommerce email testreszabás |
| **Kadence Conversions** | Popup és banner készítő |

```php title="wp-config.php"
define('WP_MEMORY_LIMIT', '256M');
```

## Teljesítmény optimalizálás

A Kadence téma és blokkok teljesítményorientáltak:

- **Dinamikus CSS betöltés** - Csak a használt stílusok töltődnek be
- **Dinamikus blokk betöltés** - Csak a használt blokkok JavaScript-je töltődik
- **Natív lazy loading** - Képek késleltetett betöltése
- **Preload támogatás** - Kritikus erőforrások előtöltése

:::tip Gyorsítótár
A Kadence kiválóan működik cache pluginokkal:
- WP Rocket
- LiteSpeed Cache
- W3 Total Cache
- FlyingPress
:::

## Starter Templates importálás

A Kadence Starter Templates importálásához:

- Stabil internetkapcsolat szükséges
- `max_execution_time` legalább 300 másodperc
- `memory_limit` legalább 256 MB

:::caution Importálási problémák
Ha a sablon importálás sikertelen, próbáld meg:
1. Növeld a `max_execution_time` értékét 600-ra
2. Kapcsold ki ideiglenesen a cache pluginokat
3. Használj másik böngészőt
:::
