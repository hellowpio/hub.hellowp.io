# <FontAwesomeIcon icon="fa-duotone fa-inboxes text-primary" /> Divi ajánlott tárhelybeállítások

Ezen az oldalon összefoglaljuk a rendszerkövetelményeket, amelyek szükségesek a Divi téma megfelelő működéséhez. Ha nem vagy biztos benne, hogy a szervered támogatja-e ezeket, kérdezd meg a tárhelyszolgáltatódat.

## PHP beállítások

A Divi megfelelő működéséhez az alábbi PHP beállításokat javasoljuk:

```ini title="php.ini"
upload_max_filesize = 128M
post_max_size = 128M
max_execution_time = 300
max_input_time = 300
max_input_vars = 10000
memory_limit = 256M
```

### Beállítás .htaccess fájlban

Ha nincs hozzáférésed a php.ini fájlhoz, a `.htaccess` fájlban is megadhatod:

```apache title=".htaccess"
php_value upload_max_filesize 128M
php_value post_max_size 128M
php_value max_execution_time 300
php_value max_input_time 300
php_value max_input_vars 10000
php_value memory_limit 256M
```

:::tip Tipp
A Divi Builder komplex oldalak esetén több erőforrást igényelhet. Ha "504 Gateway Timeout" vagy "Maximum execution time exceeded" hibákat kapsz, növeld a `max_execution_time` értékét 600-ra.
:::

### Bizonytalan vagy? Nem tudod, hogy ezt hogyan kell beállítani?

Semmi gond, a szolgáltatód tud segíteni. Küldd el nekik az alábbi levelet:

> Tisztelt Szolgáltatóm,
>
> Kérem, állítsák be a domain.hu *(írd át a sajátodra)* tárhelyhez tartozó PHP limiteket az alábbiakra:
>
> - upload_max_filesize: 128M
> - post_max_size: 128M
> - max_execution_time: 300
> - max_input_time: 300
> - max_input_vars: 10000
> - memory_limit: 256M
>
> Indoklás: A WordPress és Divi megfelelő működéséhez ezekre a beállításokra van szükségem.
>
> Köszönöm!

## WordPress verzió

**WordPress 6.4 vagy újabb** verzió szükséges a Divi legújabb verziójához.

:::info Ajánlás
Mindig a legújabb WordPress verziót használd a legjobb kompatibilitás és biztonság érdekében.
:::

## PHP verzió

**PHP 8.1 vagy újabb** verzió ajánlott.

| PHP verzió | Támogatás |
|------------|-----------|
| PHP 8.3 | Teljes támogatás |
| PHP 8.2 | Teljes támogatás |
| PHP 8.1 | Teljes támogatás |
| PHP 8.0 | Támogatott, de nem ajánlott |
| PHP 7.4 | Elavult, frissítés szükséges |

:::warning Fontos
A PHP 7.4 biztonsági támogatása 2022 novemberében megszűnt. Erősen javasolt a PHP 8.1 vagy újabb verzióra való frissítés.
:::

## Adatbázis

- **MySQL 8.0** vagy újabb (ajánlott)
- **MySQL 5.7** minimum
- **MariaDB 10.4** vagy újabb

## Webszerver

A Divi az alábbi webszervereken működik:

- **Apache 2.4** (mod_rewrite szükséges)
- **Nginx**
- **LiteSpeed**

## További ajánlások

### HTTPS

A Divi és a WordPress is HTTPS használatát javasolja. Győződj meg róla, hogy SSL tanúsítvány telepítve van.

### PHP bővítmények

Az alábbi PHP bővítmények szükségesek:

- curl
- dom
- exif
- fileinfo
- gd vagy imagick
- json
- mbstring
- openssl
- xml
- zip

### Gyorsítótár

A Divi jól működik az alábbi cache megoldásokkal:

- WP Rocket
- W3 Total Cache
- LiteSpeed Cache

:::caution Figyelem
A Divi Builder szerkesztési módban kikapcsolt cache-sel működik a legjobban. Győződj meg róla, hogy a cache plugin kizárja a bejelentkezett felhasználókat.
:::
