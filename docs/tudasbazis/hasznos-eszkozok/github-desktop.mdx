# <FontAwesomeIcon icon="fa-brands fa-github text-primary" /> GitHub Desktop - Vizuális Git kezelő

A **GitHub Desktop** egy ingyenes, nyílt forráskódú alkalmazás, amely lehetővé teszi a Git verziókezelés használatát grafikus felületen keresztül. Ideális kezdőknek és azoknak, akik vizuálisan szeretnék kezelni a projektjeiket.

:::tip Miért használd?
- **Egyszerű kezelőfelület** - Git parancsok nélkül
- **Vizuális diff nézet** - Változások könnyen áttekinthetők
- **GitHub integráció** - Zökkenőmentes együttműködés
- **Cross-platform** - Windows és macOS támogatás
:::

---

## Főbb funkciók

### Verziókezelés vizuálisan

- **Commit** - Változások mentése leíró üzenetekkel
- **Branch kezelés** - Ágak létrehozása, váltása, összevonása
- **Diff nézet** - Fájlok változásainak vizuális összehasonlítása
- **History** - Commit előzmények böngészése

### Együttműködés

- **Pull Request** - PR-ek létrehozása és kezelése
- **Clone** - Repository-k klónozása egy kattintással
- **Push/Pull** - Szinkronizálás a távoli szerverrel
- **Conflict resolution** - Merge konfliktusok feloldása

### Haladó funkciók

- **Cherry-pick** - Commit-ok másolása ágak között
- **Rebase** - Commit history átrendezése
- **Stash** - Változások ideiglenes elmentése
- **Drag & drop** - Commit-ok átrendezése húzással

## Telepítés

### Letöltés

1. Látogass el a hivatalos oldalra: [desktop.github.com/download](https://desktop.github.com/download/)
2. Válaszd ki a platformodat:
   - **macOS** - Apple Silicon és Intel támogatás
   - **Windows** - 64-bit
3. Töltsd le és futtasd a telepítőt

### Rendszerkövetelmények

| Platform | Követelmény |
|----------|-------------|
| **macOS** | macOS 10.15+ (Catalina vagy újabb) |
| **Windows** | Windows 10/11 64-bit |

## Első lépések

### 1. GitHub fiók csatlakoztatása

1. Indítsd el a GitHub Desktop-ot
2. Kattints a **Sign in to GitHub.com** gombra
3. Jelentkezz be a böngészőben
4. Engedélyezd az alkalmazást

### 2. Repository klónozása

1. **File → Clone Repository**
2. Válaszd ki a repository-t a listából
3. Állítsd be a helyi mappát
4. Kattints a **Clone** gombra

### 3. Változások commitálása

1. Végezd el a módosításokat a fájlokban
2. A GitHub Desktop automatikusan mutatja a változásokat
3. Írd be a commit üzenetet (vagy használd a Copilot-ot)
4. Kattints a **Commit to main** gombra

### 4. Push a szerverre

1. Kattints a **Push origin** gombra
2. A változások feltöltődnek a GitHub-ra

## Hasznos billentyűkombinációk

| Kombináció | Művelet |
|------------|---------|
| `Cmd/Ctrl + Shift + N` | Új repository |
| `Cmd/Ctrl + O` | Repository megnyitása |
| `Cmd/Ctrl + Enter` | Commit |
| `Cmd/Ctrl + P` | Push |
| `Cmd/Ctrl + Shift + P` | Pull |
| `Cmd/Ctrl + B` | Branch lista |
