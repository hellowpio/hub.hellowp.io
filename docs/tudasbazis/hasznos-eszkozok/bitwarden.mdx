## Első lépések

### 1. Fiók létrehozása

1. Látogass el: [bitwarden.com](https://bitwarden.com/)
2. Kattints a **Get Started** gombra
3. Add meg az email címed
4. Hozz létre egy **erős master jelszót**

:::warning Master jelszó
A master jelszót **SOHA** nem lehet visszaállítani. Válassz olyat, amit megjegyzel, de más nem tudja kitalálni!
:::

### 2. Böngésző extension telepítése

1. Telepítsd a megfelelő extension-t
2. Jelentkezz be a Bitwarden fiókodba
3. Engedélyezd az automatikus kitöltést

### 3. Jelszavak importálása

1. Menj a **Web Vault** → **Tools** → **Import Data**
2. Válaszd ki a forrást (Chrome, LastPass, 1Password, stb.)
3. Töltsd fel az export fájlt
4. Ellenőrizd az importált adatokat

### 4. Jelszó generálás

1. Kattints a Bitwarden ikonra
2. Válaszd a **Generator** fület
3. Állítsd be a paramétereket:
   - Hossz (ajánlott: 16+)
   - Kis/nagybetűk
   - Számok
   - Speciális karakterek
4. Kattints a **Regenerate** gombra
5. Másold ki és használd

## Biztonsági funkciók

### Kétfaktoros hitelesítés (2FA)

A Bitwarden fiók védelméhez beállítható:
- **Authenticator app** (TOTP)
- **Email kód**
- **YubiKey** (Premium)
- **FIDO2 WebAuthn** (Premium)

### Vault timeout

Beállítható, hogy mikor zárja le automatikusan a vault-ot:
- Böngésző bezárásakor
- Adott idő után (1 perc - 4 óra)
- Rendszer zároláskor

### Emergency Access (Premium)

Megbízható személy hozzáférhet a vault-odhoz vészhelyzet esetén, beállított várakozási idő után.

## Self-hosting

A Bitwarden teljesen self-hostolható:

### Rendszerkövetelmények

- Docker és Docker Compose
- 2GB RAM minimum
- 10GB tárhely
- HTTPS tanúsítvány

### Telepítés

```bash
# Bitwarden telepítő letöltése
curl -Lso bitwarden.sh "https://func.bitwarden.com/api/dl/?app=self-host&platform=linux"
chmod +x bitwarden.sh

# Telepítés
./bitwarden.sh install

# Indítás
./bitwarden.sh start
```
