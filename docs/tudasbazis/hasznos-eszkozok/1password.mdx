# <FontAwesomeIcon icon="fa-duotone fa-key text-primary" /> 1Password - Prémium jelszókezelő

Az **1Password** az egyik legmegbízhatóbb jelszókezelő, amelyet több mint 150 000 vállalat használ világszerte, köztük a Slack, Salesforce és IBM. 2005 óta működik és soha nem szenvedett biztonsági incidenst.

:::tip Miért válaszd az 1Password-öt?
- **Kiváló felület** - Intuitív, szép design
- **Watchtower** - Real-time biztonsági figyelmeztetések
- **Travel Mode** - Biztonságos határátkelés
- **Passkey támogatás** - Jelszó nélküli jövő
:::

## Főbb funkciók

### Jelszókezelés

- **Korlátlan jelszavak** - Minden eszközön
- **Automatikus kitöltés** - Böngésző, mobil, desktop
- **Jelszó generátor** - Testreszabható erős jelszavak
- **Categories** - Login, kártya, dokumentum, stb.

### Biztonság

- **AES-256 titkosítás** - Iparági szabvány
- **Secret Key** - 34 karakteres egyedi kulcs
- **Zero-knowledge** - Senki más nem látja az adataidat
- **2FA** - Kétfaktoros hitelesítés támogatás
- **Rendszeres audit** - Független biztonsági ellenőrzések

### Travel Mode

Utazáskor elrejtheted a bizalmas vault-okat:
- Csak a kijelölt vault-ok látszanak
- Határátkeléskor biztonságosabb
- Egy kattintással be/ki kapcsolható

### Vault megosztás

- **Családi megosztás** - 5 családtag együtt
- **Csapat vault-ok** - Üzleti felhasználóknak
- **Guest accounts** - Külsős hozzáférés
- **Jogosultságok** - Olvasás/írás szabályozás

## Első lépések

### 1. Fiók létrehozása

1. Látogass el: [1password.com](https://1password.com/)
2. Válassz csomagot és kattints a **Try free** gombra
3. Add meg az adataidat
4. Hozz létre egy **master jelszót**
5. Mentsd el az **Emergency Kit**-et biztonságos helyre

:::warning Emergency Kit
Az Emergency Kit tartalmazza a Secret Key-t és a master jelszavadat. **Nyomtasd ki és tárold biztonságos helyen!**
:::

### 2. Secret Key

A Secret Key egy 34 karakteres kulcs, amely:
- Csak a te eszközeiden van tárolva
- Új eszközön szükséges a bejelentkezéshez
- Extra biztonsági réteg a master jelszó mellett

### 3. Alkalmazások telepítése

1. Telepítsd a desktop alkalmazást
2. Telepítsd a böngésző extension-t
3. Telepítsd a mobil alkalmazást
4. Jelentkezz be mindenhol

### 4. Jelszavak importálása

1. Nyisd meg az 1Password desktop alkalmazást
2. **File** → **Import** → **Válaszd ki a forrást**
3. Töltsd fel az export fájlt
4. Ellenőrizd és rendszerezd

## Watchtower

A Watchtower folyamatosan figyeli a biztonságot:

### Mit ellenőriz?

| Kategória | Leírás |
|-----------|--------|
| **Compromised** | Kiszivárgott jelszavak |
| **Reused** | Többször használt jelszavak |
| **Weak** | Gyenge jelszavak |
| **2FA available** | 2FA beállítható, de nincs |
| **Expiring** | Lejáró elemek |
| **HTTP** | Nem biztonságos oldalak |

### Hogyan használd?

1. Nyisd meg az 1Password-öt
2. Kattints a **Watchtower** fülre
3. Nézd át a figyelmeztetéseket
4. Javítsd a problémás jelszavakat

## Developer funkciók

### SSH Key kezelés

1Password képes SSH kulcsok tárolására és használatára:

```bash
# SSH agent beállítása
export SSH_AUTH_SOCK=~/Library/Group\ Containers/2BUA8C4S2C.com.1password/t/agent.sock

# Git commit signing
git config --global gpg.format ssh
git config --global user.signingkey "ssh-ed25519 AAAAC..."
```

### CLI

```bash
# Telepítés
brew install 1password-cli

# Bejelentkezés
op signin

# Secret olvasása
op read "op://Private/API Key/credential"

# Környezeti változó injection
op run --env-file=.env -- npm start
```
