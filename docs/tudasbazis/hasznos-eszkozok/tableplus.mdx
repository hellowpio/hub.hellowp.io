# <FontAwesomeIcon icon="fa-duotone fa-database text-primary" /> TablePlus - Modern adatbázis kezelő

A **TablePlus** egy modern, natív adatbázis-kezelő alkalmazás intuitív GUI felülettel. Gyors, megbízható és támogatja a legnépszerűbb adatbázis-motorokat.

:::tip Miért válaszd a TablePlus-t?
- **Natív alkalmazás** - Gyors és megbízható minden platformon
- **Szép felület** - Modern, letisztult design
- **Multi-database** - Egy eszközzel több adatbázis típus
- **Ingyenes verzió** - Korlátozott, de nem lejáró próba
:::

## Első lépések

### 1. Kapcsolat létrehozása

1. Kattints a **Create a new connection** gombra
2. Válaszd ki az adatbázis típusát
3. Add meg a kapcsolódási adatokat:
   - **Host** - pl. `localhost` vagy `127.0.0.1`
   - **Port** - pl. `3306` MySQL-hez
   - **User** - Felhasználónév
   - **Password** - Jelszó
   - **Database** - Adatbázis neve
4. Kattints a **Test** gombra az ellenőrzéshez
5. Ha sikeres, kattints a **Connect** gombra

### 2. SSH tunnel beállítása

Távoli szerverhez SSH tunnelen keresztül:

1. Kattints az **SSH** fülre a kapcsolat ablakban
2. Pipáld be az **Over SSH** opciót
3. Add meg az SSH adatokat:
   - **SSH Host** - Szerver címe
   - **SSH User** - SSH felhasználó
   - **SSH Password** vagy **SSH Key** - Hitelesítés

### 3. Query futtatása

1. Nyisd meg a SQL szerkesztőt: `Cmd/Ctrl + E`
2. Írd be a query-t:

```sql
SELECT * FROM users WHERE active = 1 LIMIT 10;
```

3. Futtasd: `Cmd/Ctrl + Enter`

## WordPress fejlesztéshez

A TablePlus kiválóan használható WordPress adatbázisok kezeléséhez:

### Hasznos lekérdezések

```sql
-- Összes bejegyzés listázása
SELECT ID, post_title, post_status, post_date
FROM wp_posts
WHERE post_type = 'post'
ORDER BY post_date DESC;

-- Felhasználók keresése
SELECT ID, user_login, user_email, user_registered
FROM wp_users;

-- Options keresése
SELECT * FROM wp_options
WHERE option_name LIKE '%siteurl%';
```
