# <FontAwesomeIcon icon="fa-duotone fa-pen-nib text-primary" /> Affinity Suite - Adobe alternatíva

Az **Affinity** egy professzionális kreatív szoftvercsalád, amely kiváló alternatívát kínál az Adobe Creative Cloud előfizetéses modelljével szemben. 2024-ben a **Canva felvásárolta** az Affinity-t, de a szoftverek továbbra is elérhetők egyszeri díjjal.

:::tip Miért válaszd az Affinity-t?
- **Egyszeri fizetés** - nincs havi előfizetés
- **Professzionális funkciók** - Adobe-szintű képességek
- **Natív Apple Silicon támogatás** - M1, M2, M3, M4, M5 chipek
- **PSD, AI, PDF kompatibilitás** - zökkenőmentes átállás
:::

---

## Az Affinity szoftverek

### <FontAwesomeIcon icon="fa-duotone fa-image" /> Affinity Photo

Professzionális **képszerkesztő és retusáló** szoftver - Adobe Photoshop alternatíva.

**Főbb funkciók:**
- RAW képfeldolgozás
- Nem-destruktív szerkesztés
- HDR merge és focus stacking
- Fejlett retusálási eszközök
- 360°-os kép szerkesztés
- PSD fájlok teljes támogatása

### <FontAwesomeIcon icon="fa-duotone fa-vector-square" /> Affinity Designer

**Vektorgrafikai és UI tervező** szoftver - Adobe Illustrator alternatíva.

**Főbb funkciók:**
- Vektor és raszter munkatér egyben
- Szimbólumok és eszközök
- UI/UX tervezési eszközök
- SVG, AI, EPS export
- Korlátlan rajztáblák
- Valós idejű előnézet

### <FontAwesomeIcon icon="fa-duotone fa-book-open" /> Affinity Publisher

**Kiadványszerkesztő** szoftver - Adobe InDesign alternatíva.

**Főbb funkciók:**
- Tördelés és tipográfia
- Master oldalak
- Tartalomjegyzék generálás
- IDML import
- Nyomdakész PDF export
- StudioLink (Photo és Designer integráció)

## Rendszerkövetelmények

### macOS

| Követelmény | Minimum |
|-------------|---------|
| **macOS verzió** | macOS 10.15 (Catalina) vagy újabb |
| **Chip támogatás** | Apple Silicon (M1-M5) és Intel |
| **Eszközök** | MacBook, MacBook Pro, MacBook Air, iMac, Mac mini, Mac Studio, Mac Pro |
| **RAM** | 4 GB (8 GB+ ajánlott) |
| **Tárhely** | 3.5 GB szabad hely |
| **Kijelző** | 1280×768 minimum |

### Windows

| Követelmény | Minimum |
|-------------|---------|
| **Windows verzió** | Windows 10 vagy 11 (64-bit) |
| **Processzor** | x64 alapú processzor SSE2 támogatással |
| **RAM** | 4 GB (8 GB+ ajánlott) |
| **Tárhely** | 2.5 GB szabad hely |
| **Grafika** | DirectX 10 kompatibilis GPU |
| **Kijelző** | 1280×768 minimum |
